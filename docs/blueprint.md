# **App Name**: CleanSweep Subscriptions

## Core Features:

- Landing Page Display: Showcase subscription plans, benefits, and FAQs on a modern, visually appealing landing page.
- OTP Authentication: Secure user login using OTP verification for accessing the customer dashboard.
- Subscription Management Dashboard: Provide a dashboard for users to manage their subscriptions, payment methods, and booking history.
- API Integration: Connect to the Cleaning Service API to fetch and display product details, and manage subscriptions.
- Payment Processing with Stripe: Securely handle payments and manage payment methods using Stripe integration.
- Dynamic FAQ Generation: Generate static  FAQ section on the landing page, addressing common subscription questions and concerns.

## Style Guidelines:

- Primary color: Deep blue (#2D5C9F) for a professional and trustworthy feel, referencing the cleanliness of water and the depth of quality.
- Background color: Light blue (#E0F0FF) to provide a clean and calming backdrop, subtly hinting at the app's purpose.
- Accent color: Soft violet (#9F5F9F) to add a touch of sophistication and luxury, contrasting gently with the primary color.
- Body and headline font: 'Inter', a sans-serif font known for its modern, neutral, and highly readable design; suitable for both headlines and body text.
- Use Lucide React icons for a consistent and modern look across the platform.
- Mobile-first, responsive design with a clean, intuitive layout that scales well across devices.
- Subtle fade-in and slide-in animations to enhance user experience without being intrusive.
